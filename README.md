# MARPRO VHF Radio Simulator

**Interactive VHF Marine Radio Simulator**  
Built for **LROCP (Long Range Operator Certificate of Proficiency)** marine radio training.

This is a fully functional, browser-based simulator designed by MARPRO to help students practise real VHF radio operations without needing physical equipment.

### Features

- Working **Volume** and **Squelch** dials (drag to turn — just like the real radio)
- Channel selection via numeric keypad, Chan ↑/↓, or direct 16/C button
- **DSC Distress** button — hold for 3 seconds (with countdown and beeps)
- PTT (Press to Transmit) with simulated voice or text input
- Background radio static that responds to squelch and volume settings
- Dual Watch, HI/LO power, attenuator, weather mode, and more
- Fully responsive — works great on phones, tablets, and desktops
- Can be added to home screen as a Progressive Web App (PWA)

### How to Use

1. Open the simulator link.
2. Click **Power On/Off** to start the radio.
3. **Drag the Volume and Squelch dials** up/down to adjust (you will hear beeps and static).
4. Use the keypad or channel buttons to change frequency.
5. Hold the red **DSC Distress** button for 3 seconds to simulate an emergency alert.
6. Press and hold **PTT** to transmit (use voice or type a message).
7. Practise Mayday, Pan-Pan, Securité, and routine calls.

Perfect for self-paced practice or live training sessions with your MARPRO instructor.
